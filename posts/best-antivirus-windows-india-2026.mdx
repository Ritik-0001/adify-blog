## Windows Defender — Free

**Best free antivirus.** Windows Defender (Microsoft Defender Antivirus) is built into Windows 10 and 11 — no installation required. Independent lab tests consistently rate Defender at 99.5%+ detection rates, comparable to paid alternatives. For users who avoid pirated software and practise basic security hygiene, Defender is sufficient.

Defender's weakness: it does not include VPN, password manager, or dark web monitoring features included with paid suites.

## Indian Threat Landscape

India-specific threats include WhatsApp-spread APK files containing malware, fake UPI payment apps, and bank phishing sites designed for Indian payment systems. Paid antiviruses with India-focused threat databases (Quick Heal, Kaspersky with Indian data) detect regional threats faster than global products tuned primarily for Western threat profiles.
