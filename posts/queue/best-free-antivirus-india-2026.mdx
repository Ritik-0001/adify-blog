## Windows Defender — Free (Windows 10/11)

**Best free Windows antivirus.** Microsoft Defender is silently active on every Windows 10/11 PC without any user action. Regular automatic updates via Windows Update keep its threat database current. The latest AV-TEST results show Defender at 99.5% detection — competitive with paid alternatives. No ads, no upselling, no performance impact beyond what Microsoft already builds in.

For Indian users running genuine Windows 10/11 on modern hardware, Defender + browser-based ad blocking (uBlock Origin extension) provides comprehensive protection for zero cost.

## Avast Free Antivirus — Free

**Best free option if you need more features.** Avast's free tier includes real-time protection, Wi-Fi Inspector (checks your network for vulnerabilities), and a Software Updater that flags outdated apps. Detection rate is 99.7%. The trade-off is significant upselling — Avast displays frequent upgrade prompts and installs Avast One components that require active management to remove.

---

## Malwarebytes Free — Scan-Only

**Best for secondary scanning.** Malwarebytes Free does not provide real-time protection — it only scans when you manually trigger it. However, its malware detection database catches threats that Windows Defender occasionally misses. Use Malwarebytes Free alongside Windows Defender for periodic deep scans rather than as a Defender replacement.

## Do Indian Users Need Paid Antivirus?

For users who: (1) only download software from official sources, (2) don't click email attachments from unknown senders, (3) avoid pirated software — Windows Defender is sufficient. For users who regularly share USB drives with others, download from grey market sites, or open documents from WhatsApp contacts — a paid antivirus (₹850–1300/year) provides meaningful additional protection.
