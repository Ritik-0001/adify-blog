## BSNL Router Compatibility — What You Need to Know

BSNL operates two types of broadband connections in India, and the router you need depends on which one you have:

**BSNL FTTH (Fiber to the Home):**
BSNL FTTH delivers fiber to your home through an ONT (Optical Network Terminal) device. The ONT converts the fiber signal to Ethernet. Your router connects to the ONT via a standard LAN cable (RJ45). This means **any router with a WAN port** works with BSNL FTTH — you just need to enter your PPPoE credentials.

**BSNL ADSL/VDSL (Copper-based broadband):**
BSNL's older copper-based plans use ADSL or VDSL technology. Your router must either have a **built-in ADSL modem** (a router-modem combo), or you need a separate ADSL modem connected to a standard router. The VPI and VCI settings (usually 0/35 for India) must be entered correctly.

**Key setting for BSNL ADSL:**
- VPI: 0
- VCI: 35
- Protocol: PPPoE or PPPoA (check with your local BSNL office — it varies by exchange)

## How to Configure TP-Link Archer C6 for BSNL FTTH

**Step 1: Physical connection**
Connect BSNL ONT's LAN1 port → TP-Link Archer C6's WAN (blue) port using the supplied ethernet cable.

**Step 2: Access router web interface**
Open a browser and go to **192.168.0.1** or **tplinkwifi.net**. Default login: admin/admin (change this immediately).

**Step 3: Run Quick Setup**
Click Quick Setup → Select your time zone (India/Kolkata) → Choose "PPPoE" as the connection type.

**Step 4: Enter BSNL credentials**
- Username: your BSNL username (provided at the time of connection, e.g., 9412345678@bsnl.in or sometimes just the phone number)
- Password: your BSNL PPPoE password (call BSNL 1500 if you don't have this)

**Step 5: Configure WiFi**
Set your 2.4GHz and 5GHz network names (SSIDs) and passwords. Save and reboot.

**Step 6: Test connection**
The WAN IP should populate within 30–60 seconds if credentials are correct. If not, double-check the username format with BSNL.

## Common BSNL Router Problems and Fixes

**Problem: PPPoE authentication fails**
- Double-check the username format — BSNL uses different formats in different cities (phone_number@bsnl.in, username@bsnl.net, etc.)
- Call BSNL 1500 to confirm your exact PPPoE credentials
- Try leaving the password blank and just entering the username — some older BSNL connections don't use passwords

**Problem: Router connects but internet doesn't work**
- Check DNS settings — set Primary DNS to 8.8.8.8 (Google) or 1.1.1.1 (Cloudflare) manually
- BSNL's automatic DNS sometimes has issues; manual DNS resolves this immediately

**Problem: Slow speeds despite fast BSNL plan**
- Run a speed test on a **wired connection** first to confirm the BSNL line speed
- If wired is fast but WiFi is slow, move the router to a more central location or switch to the 5GHz band
- Check if the router's WAN port is Gigabit — a 100Mbps WAN port caps speeds on 200Mbps+ plans

**Problem: Router keeps disconnecting**
- Check the physical cable between the ONT and router — replace if worn
- In the router settings, set the connection type to "Always On" rather than "Connect on Demand"
- Update the router firmware through the web interface

**Problem: BSNL ONT shows connected but no WAN IP on router**
- Power cycle the ONT first (unplug for 30 seconds), then reboot the router
- Some BSNL ONTs only give one active LAN port — use LAN1, not LAN2/3/4
