## What Specs Matter for Coding

**RAM:** 16GB minimum. Docker + IDE + browser = 12–14GB easily consumed. 32GB if you run ML models or large databases locally.

**CPU:** Single-core speed matters for IDE responsiveness; multi-core matters for builds and compilation. Apple M-series chips lead both categories.

**SSD:** NVMe SSD is mandatory — spinning HDD will make every git operation and IDE load painfully slow. 512GB minimum.

**Display:** 1080p IPS minimum. OLED or Apple Retina if you code for 8+ hours — your eyes will thank you.

**Battery:** Critical for Indian developers. Power cuts and unreliable office charging mean you need 8hr+ real-world battery.

---

## Windows vs MacBook for Coding

**Choose MacBook if:** You do iOS/macOS development, work with Unix tools natively, or prioritise battery and display.

**Choose Windows if:** You need Windows-only tools, game on the side, or your budget is under ₹80,000.
